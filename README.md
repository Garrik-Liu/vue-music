# Vue-Music 音乐应用

> 本项目是基于 "Vue 全家桶" + "QQ 音乐接口" 的移动端音乐 App

##  应用入口

[Vue Music 音乐应用](http://www.garrikliu.com)

注意: 本项目为移动端应用, 请用手机浏览器打开. 目前只兼容 **火狐浏览器**.  桌面端浏览的话, 请用开发工具将视口调至移动端尺寸.

## 项目构建

```bash
# 克隆项目库
git clone git clone https://github.com/poetries/vue-music.git

# 安装依赖
npm install

# 开发环境构建
npm run dev

# 生产环境构建
npm run build
```
